pip_init
==========

**pip_init** は、Pythonに :code:`npm init` や :code:`swift init` のようなパッケージ構成CLIを提供するライブラリです。

デフォルトでは pip用に最適化されたテンプレートが用意されていますが、必要に応じて自由にカスタマイズできます。
また、各テンプレートはPythonモジュールとして読み込まれるので、pip_initそのものの動作を部分的に変更することも可能です。

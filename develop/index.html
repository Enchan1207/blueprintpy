blueprintpy

Overview

blueprintpy は、npm init や swift init のようなパッケージ構成CLIを生成するユーティリティです。

パッケージ構成は Jinja2を使用したテンプレートで管理され、自由に作成・カスタマイズできます。 デフォルトではpypi準拠のPythonパッケージおよびCMake準拠のC++プロジェクト用CLIが用意されていますが、 今後の更新によって随時追加していく予定です。 もちろん、ユーザの皆様からのPRも歓迎です。

Installation

blueprintpyはPyPIで公開されています。パッケージとともにCLIツール blueprint がインストールされます。

pip install blueprintpy
